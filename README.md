# BigBinary-cr-ation-d-une-structure-et-manipulation-de-grands-nombres-binaires-en-C
Implémentation en c d’une structure (BigBinary) permettant de manipuler des nombres binaires de très grande taille. Le projet propose des opérations arithmétiques et logiques (addition, soustraction, division par 2, modulo, pgcd, comparaisons) avec une gestion manuelle de la mémoire.
